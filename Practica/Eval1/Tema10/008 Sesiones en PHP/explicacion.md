# Indroduccion brece y contexalizacion
En nuestra sociedad actual, los videojuegos y las películas son dos formas populares de entretenimiento que atraen a muchas personas, incluyendo estudiantes como tú. A menudo, estos hobbies se combinan con otras actividades diarias como nadar o seguir el fútbol. Hoy, vamos a explorar cómo podemos utilizar variables en PHP para manejar información sobre nuestros hobbies.


# Desarrollo técnico correcto y preciso
## declara php
```
<?php

?>
```
## declare variables `hobby_preferido` y `tiempo_libre_hobbies`
```
$hobby_preferido = "jugar_video_juego";

$tiempo_libre_hobbies = 15;
```
## demustra en la pantalla
```
echo $hobby_preferido 

echo $tiempo_libre_hobbies

```

# Codigo completa

```
<?php
$hobby_preferido = "jugar_video_juego";

$tiempo_libre_hobbies = 15;

echo $hobby_preferido 

echo $tiempo_libre_hobbies
```

# Cierre/Conclusión enlazando con la unidad
Este ejercicio demuestra cómo las variables en PHP nos permiten almacenar y mostrar información relacionada con situaciones reales, como nuestros hobbies y el tiempo que les dedicamos. Comprender el uso de variables es esencial, ya que son la base para trabajar con datos en programación y nos acompañarán a lo largo de toda la unidad y en futuros proyectos más complejos.
